Conceptos de HTML, CSS y JavaScript

HTML

¿Qué es una etiqueta HTML?

Una etiqueta HTML define la estructura y contenido de una página web. Ejemplo: <p>Texto</p> crea un párrafo.

HTML

Atributo HTML

Los atributos añaden información a las etiquetas. Ejemplo: <img src="imagen.jpg" alt="Descripción">.

CSS

¿Qué es CSS?

CSS estiliza páginas web, controlando diseño y apariencia. Ejemplo: color: blue;

CSS

Selector CSS

Un selector CSS apunta a elementos HTML para aplicar estilos. Ejemplo: p { font-size: 16px; }

JavaScript

¿Qué es JavaScript?

JavaScript agrega interactividad a las páginas web. Ejemplo: alert("¡Hola!");

JavaScript

Función en JavaScript

Una función es un bloque de código reutilizable. Ejemplo: function suma(a, b) { return a + b; }
